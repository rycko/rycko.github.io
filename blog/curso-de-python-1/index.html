Curso de python 1

Primera lección sobre el lenguaje de programación Python.

Jan 17, 2020 2 min de lectura programación

Python es un lenguaje de programación que se ha hecho muy popular en los últimos años, ya que es un lenguaje de programación muy potente con una amplia variedad de aplicaciones y tiene una curva de aprendizaje baja, es decir que es relativamente facil de aprender.

Instalación.

La instalación de este lenguaje es muy sencilla.

En Windows solo tienes que descargar el instalador, desde aquí lo puedes hacer, solo asegurate de descargar la versión correcta para tu sistema operativo, después de esto solo tienes que darle doble click y seguir las instrucciones.

En OSX lo primero que debes hacer es verificar que tu sistema operativo acepte paquetes que NO están en la app store, después debes bajar el repositorio, este es el enlace, por último solo debes asegurarte de descargar la versión correcta para tu sistema operativo y darle doble click al instalador.

En Ubuntu lo más probable es que ya este instalado por default, si quieres verificarlo solo debes colocar el siguiente comando: 

$ python3 --version
Si no lo tienes instalado te dejo a continuación una lista de comandos para instalarlo en diferentes distribuciones de Linux.

Debian, Ubuntu. 

$ sudo apt install python3
Fedora 
$ sudo dnf install python3
OpenSUSU 
$ sudo zypper install python3
Para verificar la instalación exitosa de Python ingresa el siguiente comando en la terminal. 
$ python3 --version

Hola Mundo en Python en modo interactivo

El modo interactivo en Python es cuando los comandos son leídos desde la terminal o linea de comandos. Para utilizar Python de esta manera lo único que debemos hacer es ingresar el comando: 

$ python3.6
te aparecerá un prompt con tres signos “Mayor que” (>), después solo tienes que inicializar una variable puedes nombrarla como tú quieras en este caso la nombramos miPrimerCodigoPy

>>>miPrimerCodigoPy=True
Entonces pones una condicion if de la siguiente manera: 
>>>if miPrimerCodigoPy:

Te apareceran tres puntos(…) indicando un subprompt para ingresar la siguiente sentencia es importante tomar en cuenta que es la sentencia de la condición “If” por lo tanto debe ir identada. 

...		print("¡Hola Mundo Python!")
te apareceran otra vez los tres puntos entonces solo debes volver a dar click y “¡Hola Mundo Python!”

Para salir de Python debes ingresar: 

>>>exit()

Esto fue tu primer codigo en Python, saludos.
