Empezando con Kubernetes, instalación de Microk8s

enero - 10 - 2020 | 1 minuto(s) para leerlo

Kubernetes es una plataforma open source que facilita la automatización de procesos en linux.

Puede crear un grupo de servidores que se comportan como uno y hostear contenedores de linux, entonces Kubernetes ayuda a administrar de manera facil y eficaz estos servidores.

Microk8s es un paquete para poder instalar Kubernetes en una computadora, está pensado para desarrolladores, para poder tener un entorno donde desplegar tus proyectos de manera local.

Instalación de Microk8s

Para poder instalarlo necesitas una computadora con el sistema operativo que admita el comando snapd.

Es recomendable un equipo con 4gb de ram por lo menos.

1.- Instalamos Microk8s desde la terminal con este comando: 

$sudo snap install microk8s  --classic
2.- Debes unirte al grupo creado por microk8s. 
$sudo usermod -a -G microk8s $USER
3.- Checa el estatus de tu instalación. 
$microk8s.status --wait-ready
4.- Puedes acceder a los nodos de la siguiente manera: 
$microk8s.kubectl get nodes
5.- También puedes acceder a los servicios. 
$microk8s.kubectl get service
6.- Puedes detener e iniciar micrk8s con estos comandos.

Para iniciarlo: 

$microk8s.start
Para detenerlo: 
$microk8s.stop
